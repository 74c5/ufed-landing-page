Scroll Lock Demo

Goals

Try to lock prevent the event handler from updating while the javascript is commanding a scroll.

Until now, I have been using a dummy timeout to block the scroll event handler function until scroll is finished. This is messy - inaccurate. What if the scroll takes longer over larger amounts of data? The update won't happen correctly. Actually, the remainder of the scroll is blocked and does not complete correctly.

So, we'll use a flag to block, until a target threshold is met, then we can unblock the standard handler.
